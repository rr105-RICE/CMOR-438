### How to run

1. Install dependencies (from the repo root):

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
```

2. Open any `.ipynb` in Jupyter (VS Code, JupyterLab, or classic notebook) and run the cells top-to-bottom.
